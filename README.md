# Route Engine Laravel

Calculador de frete dinâmico integrando ViaCEP, Nominatim e OpenRouteService.

## Requisitos
- PHP ^8.3
- Laravel ^11.0 | ^12.0

## Instalação

Como o pacote ainda não está no Packagist, adicione o repositório ao seu `composer.json`:

```json
"repositories": [
    {
        "type": "vcs",
        "url": "[https://github.com/seu-usuario/route-engine](https://github.com/seu-usuario/route-engine)"
    }
],
